BIENVENIDO A NUESTRO SITIO WEB SOMOS EL GRUPO 3

presentación

Al Carbón Fast Foods es un establecimiento comercial para comidas rápidas, el proyecto se desarrollo para permitir que los clientes puedan acceder de manera eficaz por medio de una página en internet aprovechando el uso de las nuevas tecnologias de la infomación.
El objetivo principal es poner al servicio todo el portafolio de productos comestibles y asi mismo contar con los canales virtuales para la venta y atención al cliente
